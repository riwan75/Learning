# Plan: Design System Folder Refactoring

## Goal
Centralize all UI styling into a `src/design-system/` folder. Refactor all existing files to use the new DS components and tokens.

## Decisions
| Decision | Choice |
|---|---|
| Scope | Full design system with tokens + variants |
| Refactor scope | All files that use the recurring patterns |
| Card primitive | Props-based with `variant` prop + className override |

## New structure

```
src/design-system/
├── tokens.ts              — colors, spacing, radius, padding constants
├── components/
│   ├── Card.tsx            — Translucent rounded card (variant: default | compact | flush)
│   ├── Pill.tsx            — Pill button (rounded-full background)
│   ├── IconButton.tsx      — Round icon button (rounded-full p-3)
│   ├── Screen.tsx          — LinearGradient + SafeAreaView wrapper (variant: clear | cloudy | rain | snow)
│   └── Input.tsx           — TextInput with rounded-xl bg-white/20 styling
├── variants.ts            — Variant configs (Card variants, Screen variants)
└── index.ts               — Barrel export
```

## Components

### `Card`
- Props: `variant?: "default" | "compact" | "flush"`, `className?: string`, `children`
- `default`: `bg-white/20 rounded-3xl p-6 mb-6`
- `compact`: `bg-white/20 rounded-3xl p-4 mb-6`
- `flush`: `bg-white/20 rounded-3xl mb-6` (no padding)
- className prop appends additional classes

### `Pill` (for unit toggle)
- Props: `className?: string`, `children`
- Renders: `bg-white/20 rounded-full`
- Used inside View flex-row

### `IconButton` (for search button)
- Props: `onPress`, `children` (icon emoji)
- Renders: `bg-white/20 rounded-full p-3` as TouchableOpacity

### `Screen`
- Props: `variant: "clear" | "cloudy" | "rain" | "snow"`, `children`
- Wraps in LinearGradient + SafeAreaView
- Colors derived from `weatherCodes.ts` VARIANT_COLORS

### `Input`
- Props: all TextInput props + `className?: string`
- Renders: `bg-white/20 rounded-xl px-4 py-3 text-white text-base`
